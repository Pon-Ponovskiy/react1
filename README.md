## Сайт с счетчиком
Сайт с двумя кнопками плюс и минус и счетчиком равном 0 при заходе на сайт
## Как установить
1. Скачать архив в с Github и открыть его через любой компелятор.
2. Открыть терминал/консоль и скачать node.js если не скачена
```
 sudo apt install nodejs
```
3.  Открыть проект написав в консоль/терминал
```
  npm start
```
## Как работает
1. Если нажать на кнопку плюс то к счетчику добавится единится
2. Если нажать на кнопку минус то у счетчика отнимится единица
